Redis 常见命令 ​

Redis是一种高性能的键值存储数据库，提供了丰富的命令集用于操作和管理数据。以下是一些常见的Redis命令，按照不同类别分类：

1. 基本操作命令 ​

  • 连接Redis服务器

    bash
    redis-cli

  • PING

    检查Redis服务器是否运行正常：

    plaintext
    PING

    返回PONG表示正常。

2. 键操作命令 ​

  • SET

    设置一个键的值：

    plaintext
    SET key value

  • GET

    获取一个键的值：

    plaintext
    GET key

  • DEL

    删除一个键：

    plaintext
    DEL key

  • EXISTS

    检查一个键是否存在：

    plaintext
    EXISTS key

  • EXPIRE

    为一个键设置过期时间（以秒为单位）：

    plaintext
    EXPIRE key seconds

  • PERSIST

    设置一个键永不过期：

    plaintext
    persist key

  • TTL

    获取一个键的剩余生存时间（以秒为单位）：

    plaintext
    TTL key

  • DBSIZE

    获取数据库的 key 的数量

    plaintext
    dbsize

  • TYPE

    返回键类型

    plaintext
    type key

    返回值： none (key不存在)； string (字符串)； list (列表)； set (集合)； zset (有序集)； hash (哈希表)

  • MOVE

    将当前数据库的 key 移动到给定的数据库 db 当中。

    plaintext
    MOVE key db

    如：把toimc移到2号数据库当中

    plaintext
    move toimc 2

  • RENAME

    更改键的名称：

    plaintext
    rename key newkey

3. 字符串操作命令 ​

  • APPEND

    向一个键的值追加字符串：

    plaintext
    APPEND key value

  • INCR

    将一个键的整数值增加1：

    plaintext
    INCR key

  • DECR

    将一个键的整数值减少1：

    plaintext
    DECR key

4. 哈希操作命令 ​

  • HSET

    设置哈希表中字段的值：

    plaintext
    HSET hash field value

  • HGET

    获取哈希表中字段的值：

    plaintext
    HGET hash field

  • HDEL

    删除一个哈希表字段：

    plaintext
    HDEL hash field

  • HGETALL

    获取哈希表中的所有字段和值：

    plaintext
    HGETALL hash

5. 列表操作命令 ​

  • LPUSH

    将一个值插入到列表的头部：

    plaintext
    LPUSH list value

  • RPUSH

    将一个值插入到列表的尾部：

    plaintext
    RPUSH list value

  • LPOP

    移出并获取列表的第一个元素：

    plaintext
    LPOP list

  • RPOP

    移出并获取列表的最后一个元素：

    plaintext
    RPOP list

  • LRANGE

    获取列表指定范围内的元素：

    plaintext
    LRANGE list start stop

6. 集合操作命令 ​

  • SADD

    向集合添加一个成员：

    plaintext
    SADD set member

  • SREM

    移除集合中的一个成员：

    plaintext
    SREM set member

  • SMEMBERS

    获取集合中的所有成员：

    plaintext
    SMEMBERS set

7. 有序集合操作命令 ​

  • ZADD

    向有序集合添加一个成员：

    plaintext
    ZADD zset score member

  • ZREM

    移除有序集合中的一个成员：

    plaintext
    ZREM zset member

  • ZRANGE

    获取有序集合中指定范围内的成员：

    plaintext
    ZRANGE zset start stop

8. 事务操作命令 ​

  • MULTI

    开始一个事务块：

    plaintext
    MULTI

  • EXEC

    执行所有事务块中的命令：

    plaintext
    EXEC

  • DISCARD

    取消事务块中的所有命令：

    plaintext
    DISCARD

9. 发布/订阅操作命令 ​

  • PUBLISH

    将消息发布到指定频道：

    plaintext
    PUBLISH channel message

  • SUBSCRIBE

    订阅指定的频道：

    plaintext
    SUBSCRIBE channel

  • UNSUBSCRIBE

    取消订阅指定的频道：

    plaintext
    UNSUBSCRIBE channel

10. 服务器管理命令 ​

  • INFO

    获取Redis服务器的信息和统计数据：

    plaintext
    INFO

  • CONFIG GET

    获取Redis服务器的配置参数：

    plaintext
    CONFIG GET parameter

  • CONFIG SET

    设置Redis服务器的配置参数：

    plaintext
    CONFIG SET parameter value

  • SAVE

    将数据保存到磁盘(会阻断进程，一般很少用，一般用BGSAVE)：

    plaintext
    SAVE

  • SHUTDOWN

    关闭Redis服务器：

    plaintext
    SHUTDOWN

这些是Redis中一些常见的命令，它们涵盖了大多数日常操作。对于更详细的命令列表和用法，可以查阅Redis官方文档或使用Redis CLI中的帮助命令，例如：

plaintext
HELP @<command-category>

或者

plaintext
COMMAND

查看所有的keys

keys *

可以查看到test开头的key

keys test*

实例 ​

当然，以下是一些常用的Redis命令以及相应的使用示例，涵盖了键操作、字符串操作、哈希操作、列表操作、集合操作、有序集合操作、发布/订阅操作、事务操作和服务器管理操作。

1. 键操作命令 ​

  • SET：设置一个键的值

    plaintext
    SET mykey "Hello"
    plaintext
    OK

  • GET：获取一个键的值

    plaintext
    GET mykey
    plaintext
    "Hello"

  • DEL：删除一个键

    plaintext
    DEL mykey
    plaintext
    (integer) 1

  • EXISTS：检查一个键是否存在

    plaintext
    EXISTS mykey
    plaintext
    (integer) 0

  • EXPIRE：为一个键设置过期时间（以秒为单位）

    plaintext
    SET mykey "Hello"
EXPIRE mykey 10
    plaintext
    OK
(integer) 1

  • TTL：获取一个键的剩余生存时间（以秒为单位）

    plaintext
    TTL mykey
    plaintext
    (integer) 9

2. 字符串操作命令 ​

  • APPEND：向一个键的值追加字符串

    plaintext
    APPEND mykey " World"
    plaintext
    (integer) 11

  • INCR：将一个键的整数值增加1

    plaintext
    SET counter 100
INCR counter
    plaintext
    OK
(integer) 101

  • DECR：将一个键的整数值减少1

    plaintext
    DECR counter
    plaintext
    (integer) 100

3. 哈希操作命令 ​

  • HSET：设置哈希表中字段的值

    plaintext
    HSET myhash field1 "foo"
    plaintext
    (integer) 1

  • HGET：获取哈希表中字段的值

    plaintext
    HGET myhash field1
    plaintext
    "foo"

  • HDEL：删除一个哈希表字段

    plaintext
    HDEL myhash field1
    plaintext
    (integer) 1

  • HGETALL：获取哈希表中的所有字段和值

    plaintext
    HGETALL myhash
    plaintext
    (empty list or set)

4. 列表操作命令 ​

  • LPUSH：将一个值插入到列表的头部

    plaintext
    LPUSH mylist "world"
LPUSH mylist "hello"

(integer) 2

    plaintext

    plaintext

  • LRANGE：获取列表指定范围内的元素

    plaintext
    LRANGE mylist 0 -1
    plaintext
